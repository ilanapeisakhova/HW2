# recipes
Управление рецептами и списком покупок. 
Дынное задание позволяет создавать ингредиенты с проверкой корректности количества, добавлять рецепты, масштабировать порции, объединять несколько рецептов в общий список покупок с автоматическим суммированием одинаковых ингредиентов, а также поддерживает диетические рецепты с указанием категории (веган, без глютена и т.д.).
